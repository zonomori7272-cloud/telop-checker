# テロップ誤字脱字チェッカー

YouTube動画編集で追加したテロップ（字幕ではなく編集で重ねたテキスト）の
誤字・脱字・表記ゆれを自動チェックし、タイムスタンプ付きで一覧表示するツールです。

---

## 必要なもの

- **Python 3.9 以上**
- **pip**（Python のパッケージ管理ツール。Python に同梱されています）
- **Anthropic API キー**（Claude API を利用するため）
  - 取得方法: https://console.anthropic.com/ でアカウントを作成し、API キーを発行してください

---

## セットアップ手順

ターミナル（Mac の場合は「ターミナル.app」）を開き、以下のコマンドを **上から順に** コピー＆ペーストして実行してください。

### 1. このフォルダに移動する

```bash
cd /path/to/telop-checker
```

> ※ `/path/to/telop-checker` の部分は、実際のフォルダの場所に置き換えてください。

### 2. 仮想環境を作成する（推奨）

```bash
python3 -m venv venv
source venv/bin/activate
```

> Windows の場合は `source venv/bin/activate` の代わりに `venv\Scripts\activate` を実行してください。

### 3. 必要なライブラリをインストールする

```bash
pip install -r requirements.txt
```

インストールには数分かかる場合があります。

### 4. API キーを設定する

`.env` ファイルを作成して API キーを書き込みます。

```bash
cp .env.example .env
```

作成された `.env` ファイルをテキストエディタで開き、
`your_api_key_here` の部分を実際の API キーに書き換えてください。

```
ANTHROPIC_API_KEY=your_api_key_here
↓（例）
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxxxx
```

---

## 起動方法

```bash
python app.py
```

起動後、ターミナルに以下のような表示が出ます。

```
 * Running on http://127.0.0.1:5000
```

ブラウザで `http://localhost:5000` を開いてください。

アプリを終了するときは、ターミナルで `Ctrl + C` を押してください。

---

## 使い方

1. **ブラウザで `http://localhost:5000` を開く**

2. **MP4 ファイルをアップロードする**
   - ファイルをドラッグ＆ドロップするか、ファイル選択ボタンをクリックして動画を選びます
   - 1GB を超えるファイルはアップロードできません

3. **「チェックを開始する」ボタンを押す**
   - 動画のフレームを1秒ごとに抽出し、テロップが変化したフレームを検出します
   - 検出したフレームを Claude Vision API に送り、テロップを読み取ります
   - 読み取ったテキストの誤字・脱字・表記ゆれを自動チェックします

4. **結果を確認する**
   - タイムスタンプ・テロップ内容・問題の有無・詳細が一覧表示されます
   - 問題がある行はピンク色でハイライトされます

5. **CSV でダウンロードする（任意）**
   - 「CSV でダウンロード」ボタンで結果を Excel 等で開けるファイルとして保存できます

---

## 注意事項

- アップロードした動画ファイルは処理後に自動で削除されます
- 動画の長さに比例して処理時間がかかります（目安：10分動画で数分〜10分程度）
- API の利用料金が発生します。Anthropic の料金ページを事前にご確認ください

---

## トラブルシューティング

| エラーメッセージ | 対処方法 |
|---|---|
| `ANTHROPIC_API_KEY が設定されていません` | `.env` ファイルの API キーを確認してください |
| `APIキーが無効です` | Anthropic コンソールで正しいキーを確認してください |
| `MP4ファイルのみ対応しています` | 動画を MP4 形式に変換してからアップロードしてください |
| `ファイルサイズが1GBを超えています` | 動画を圧縮するか、短い区間に分割してください |
